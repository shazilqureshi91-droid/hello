README - Halloween Nails Landing Page

Files:
- index.html     : Main landing page (single page)
- styles.css     : Styling for the page
- script.js      : Small JS enhancements (no analytics)
- README.txt     : This file

How to use:
1. Unzip the package and open index.html in any modern browser.
2. The CTA buttons link to the affiliate URL and open in a new tab.
3. No external images are included. Edit copy and colors in index.html and styles.css as desired.

Affiliate note:
The CTA goes to an Amazon affiliate link. The page includes a brief disclosure near the button and in the footer.
